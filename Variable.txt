註解是在程式中寫筆記的方法，compiler在編譯的時候會把註解忽略掉，也就是說是給人看的筆記，電腦不會看到。保持寫註解的好習慣，會大大提升你的程式碼的可讀性，也會讓你在寫程式的時候更有條理。有些大師主張最好的註解就是沒有註解，那是因為他們將自己訓練出非常清楚好懂的coding style，對初學者和一般人而言，多寫註解還是最好的方法。

   // 兩條斜線代表單行的註解

   /* 這個代表
      多行的
      註解     */

   /************************
    * 你會看到有人寫成這樣
    * 只是為了好看而已
    ************************/

codeblocks給註解預設的顏色很淺，如果你覺得看不清楚，可以去設定中調整。當然也可以按照喜好調整其他關鍵字的顏色。


======================= 正文開始 ==============================

電腦的工作便是對資料(data)的儲存和運算。在程式語言中，資料被儲存在變數中。變數等於是一種容器，就好比一個一個的抽屜，當我們宣告一個變數時，就是在向電腦要一個存放資料的抽屜，並同時給這個抽屜賦予一個名稱。不同的程式語言有不同的宣告變數方法。在C++中，宣告變數時需要指定變數的資料型別(data type)，並給予一個名稱。

   int a;

這樣就宣告了一個叫a的變數，型別為int(integer，整數)，也就是說這個變數是用來儲存整數的。變數被宣告後就可以把資料裝進去，也就是指定(assign)給它。

   a = 10;

這樣就在a這個變數裡存入10。如果我們這時用cout把a印出來，就會看到a的值為10。我們也可以在一開始宣告變數的時候就給它一個初始值。

   int a = 10;

這樣的效果是一樣的。變數的名稱可以自己任意取，大小寫有別，但是不可以跟C++的key word重複。譬如說不可以叫int，或叫cout，但是可以叫Int或COUT。另外開頭的字不可以是數字(1126J是不行的，但J1126就可以)，也不可以有空格或底線以外的符號。

   date of baptism   (X)
   date-of-baptism   (X)
   date_of_baptism   (O)
   dateOfBaptism     (O)

C++的資料型別主要有下列幾種：

   int               integer，整數
   float/double      浮點數，就是小數
   bool              booline，ture or false
   char              character，字元

string(字串)其實就是一排char所組成的，它並不是C++原本的型別，而是後來的人加上去的，所以在使用前要include它的library，使用方式和一般型別相同。
